# 🚀 Quick Start - Mobile-Responsive Dashboard

## 1️⃣ Install Dependencies
Open Command Prompt and run:
```bash
cd C:\Users\dears\Downloads\Sporty Fitness Dashboard UI
npm install
```

## 2️⃣ Start Development Server
```bash
npm run dev
```

You'll see:
```
VITE v6.3.5  ready in XXX ms
➜  Local:   http://localhost:5173/
```

## 3️⃣ Open in Browser
- **Desktop:** Go to http://localhost:5173
- **Mobile/Tablet:** Use your phone's browser or DevTools responsive mode (press F12 → Device Toolbar)

## 📱 What's New on Mobile

### Navigation
- Clean hamburger menu on mobile
- Full navigation on desktop
- Responsive buttons and icons

### Dashboard
- **Light, clean design** (no more dark theme)
- **One-column layout** on mobile, two-column on desktop
- **User greeting** with your name
- **Touch-friendly buttons** and sliders
- **Responsive charts** that scale with screen size

### Onboarding
- **Name input** for personalization
- **Easy-to-use sliders** for height, weight, age
- **Mobile-optimized form** that adapts to screen size
- **Clear gender selection** buttons

## 📏 Responsive Breakpoints

- **Mobile** (< 768px): Single column, optimized spacing
- **Tablet** (768px - 1024px): Transitional layout
- **Desktop** (> 1024px): Full 2-column experience

## 🔧 Build for Production

When you're ready to deploy:
```bash
npm run build
```

This creates a `dist/` folder with optimized production files.
