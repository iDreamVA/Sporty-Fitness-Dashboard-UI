## 🔧 CUSTOMIZATION EXAMPLES

### Change Colors
Edit `styles-standalone.css`:
```css
--color-accent-orange: #FF7601;  ← Change to new color
```

### Add New Sensor
Edit `app-standalone.js`:
```javascript
SENSOR_TEMPLATES.push({
    type: 'newSensor',
    label: 'New Name',
    icon: '🎯',
    color: '#YOURCOLOR'
});
```

### Adjust Mobile Grid Gap
Edit `styles-standalone.css`:
```css
.puzzle-grid {
    gap: 1rem;  ← Increase from 0.75rem
}
```
